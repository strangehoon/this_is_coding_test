# Python 
```python
S = input()
alpha = []
sum = 0
for i in S:
    if i.isalpha():
        alpha.append(i)
    else:
        sum +=int(i)
alpha.sort()

# join 연산을 사용해 리스트를 문자열로 변환
result = "".join(alpha)
if sum !=0:
    result += str(sum)

print(result)

```

문자열의 문자들을 하나씩 확인하는 작업은 O(N)이지만 파이썬의 sort 메서드는 O(NlogN)이다.

* **시간 복잡도** : O(NlogN)
