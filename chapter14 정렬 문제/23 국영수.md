# Python 
```python
import sys
n = int(input())
data = []

for i in range(n):
    name, kor, eng, math = sys.stdin.readline().split()
    data.append([name, int(kor), int(eng), int(math)])

data.sort(key = lambda x: (-x[1], x[2], -x[3], x[0]))
for i in data:
    print(i[0])
```

* **시간 복잡도** : O(nlogn)
