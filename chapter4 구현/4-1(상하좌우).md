# Python 풀이
```python
N = int(input())
data = list(input().split())
x,y = 1,1
dx = [0, 0, -1, 1]
dy = [-1, 1, 0, 0]
dir = ['L', 'R', 'U', 'D']

# data를 하나씩 확인
for i in data:
    for j in range(len(dir)):
        if i == dir[j]:
            nx = x+ dx[j]
            ny = y+ dy[j]
    # 공간을 벗어나는 경우 무시
    if 1>nx or nx > 5 or 1>ny or ny > 5:
        continue
    x,y = nx, ny
print(x,y)

```



* **시간 복잡도** : O(N)
